# Raw Data

This folder contains original, unmodified data files. **Do not edit anything here — ever.**

Raw data is the foundation of your project. If you need to clean, filter, or transform something, do it in a script and save the result to `data/processed/`. This way you always have the original to fall back on, and every transformation is documented and reproducible.

This folder is intentionally excluded from version control (see `.gitignore`). Share raw data with collaborators through a secure, agreed-upon channel — not GitHub.

---

## How to document your datasets

For every dataset you add to this folder, copy and fill out this template below:
```
## [Short descriptive name]

- **File(s):**         filename.csv
- **Downloaded:**      YYYY-MM-DD
- **Source:**          Name of database, agency, collaborator, or collection method
- **URL:**             https://...
- **Citation:**        Author et al. (Year). Title. Journal/Repository. DOI
- **License:**         e.g. CC-BY, Public Domain, Data Use Agreement
- **Contents:**        Brief description of what this dataset contains — variables, spatial extent, time range, etc.
- **Notes:**           Any caveats, known issues, or quirks to be aware of
```
