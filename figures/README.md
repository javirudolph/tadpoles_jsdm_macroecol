# Figures

This folder contains all figures generated by scripts and notebooks. Figures here should be **produced programmatically** — never manually edited after the fact. If a figure needs to look different, update the script that made it and re-run it.

This folder is excluded from version control because figures are reproducible outputs. If you have the raw data and the scripts, you can always regenerate them.

---

## Saving figures in R

Use `ggsave()` with explicit dimensions and resolution:
```r
ggsave(
  filename = here("figures", "fig01_disease_prevalence.png"),
  plot = my_plot,
  width = 6.5,
  height = 4,
  units = "in",
  dpi = 300
)
```

If you're new to plotting in R, the [Software Carpentry ggplot2 lesson](https://swcarpentry.github.io/r-novice-gapminder/08-plot-ggplot2.html) is a great place to start.
